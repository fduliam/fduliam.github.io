严格教学管理下的高中生如何提分
#

————不写作业的艺术

#

摘 要

#

随着时代的发展，社会对于人才质量的要求越来越高，选拔越来越严格，其中最突出的一个表现之一就是高考难度逐渐提升，与之而来的就是高中课业压力越来越大。尤其在山东、河南、河北等地的县级高中或地区普通高中，教学管理愈加严格，老师备课压力大，学生作业压力大。尽管这种高压教学管理取得了一定成果，但由于学生自由安排的时间被过度压缩，本人认为这也在一定程度上限制了学生”曲线提分”的可能性。本人是2020年山东某县城高考第五名，录取到复旦大学信息安全专业，后保研至本校计算机攻读硕士。高中学习生活期间，在严格的教学管理模式中，本人总结出许多如何调整个人学习计划的经验。本文命名为“不写作业的艺术”，已在一定程度上反映了方法论，下面将详细从“是什么、为什么、怎么样”的叙述逻辑来阐述这一“艺术”。

{:height=“60%” width=“60%” style=“display:block; margin-left:auto; margin-right:auto;”}

以免被误会为带偏学生，本人首先声明：

  1. 本人并非鼓励学生对抗或抱怨学校和老师的努力，而只是给出一些可能有利于学生提分的建议，如果有学生拿着这份文档去和家长、老师狡辩甚至冲突，纯属个人理解能力有问题或者需要去看心理医生；
  2. 日常作业是一个训练学生熟练掌握当日所学知识的工具，老师布置作业是必要的，学生写作业也是必要的，完全放弃作业而自学是愚蠢的；
  3. 本文方法论并不适用于所有学生，甚至可能只适用于作者自己，属于个人经验之谈，仅供参考。

一、作者“不写作业”的历史
#

第一重境界
#

本人无法按时完成所有作业（注意作者此时已不再使用“不写作业”，本文标题过于绝对，请读者见谅）的习惯似乎从小学产生端倪，彼时课间同学们都在狂写作业，期望能够少带一些书回家，而本人宁可坐在座位上发呆，因为觉得上完课还要写作业实在太累，但本人回家后其实可以保证每天完成作业。

真正逐渐无法完成作业是从初中开始。初中时本人住校，周五晚上回家，周日下午返校。初一班主任担心住校生时间短在家完不成周末作业，于是向大家声明：写不完的部分可以周日下午返校后完成。

这样其实为当时的我提供了一个偷懒的理由，于是将所有作业拖到了周末下午和晚上，而实际上这样做并不能保证每次都能按时交上作业。加上住校生活十点钟强制熄灯睡觉，舍友再聊聊天，本人写字也比较慢，导致日常作业也难以按时完成。

从初二开始，本人频繁因为无法完成作业而罚站，但因为课上听讲较为认真，而且写完的部分属于是认认真真的完成，所以本人成绩还算属优秀。

但这还只属于“不写作业的第一重境界”，称为“没有安排好时间而写不完作业”（本人虽然因为各种原因写不完作业，但总体上还较为自律，与完全无法自律所有时间都在玩并不相同，后者无法称为“境界”）。因为写不完作业而罚站的同学，我预计有相当一部分处于这个境界，原因可能是个人因为各种因素转移了注意力或者时间被占用。

处于该境界下，学生可能会产生后悔和焦虑心理，比如担心不知如何应对老师的不满，或者被罚站丢面子。但对于处于青春期的本人而言，恰恰是这样的经历，让本人在心理上习惯了写不完作业的行为，以及写不完作业带来的后果。换言之，本人写不完作业可以做到脸不红心不跳，被老师批评也可以做到左耳进右耳出（当然还是要迎合一下，不能将左耳进右耳出的姿态摆出来，不然会被误以为是百不听邪而挨揍）。

{:height=“60%” width=“60%” style=“display:block; margin-left:auto; margin-right:auto;”} 这样的经历奠定了本人进入“不写作业的第二重境界”的心理基础。

第二重境界
#

进入高中后，写不完作业的情况依然时常发生，但原因发生了变化。高中住校生活是每周校内放假一下午，每月回家两天，每晚晚自习持续到十点，宿舍十点四十五强制熄灯睡觉，所以除了基本生活需求之外，基本所有时间都用来学习了。进入高中后，个人也懂事了很多，加上当时被《平凡的世界》这本书刺激，开始从内到外的勤奋努力。

但即便如此，我依然难以每天都完成所有作业。

进入高中后，数学物理学科开始强调刷课外题，我竭尽全力下，前两个学期也只能完成课外题的三分之二左右（数学做一半，物理能做五分之四），文科作业（语文、英语、政史地）大概有三分之一的情况下完不成，理科作业大概每周完不成一两次。

{:height=“60%” width=“60%” style=“display:block; margin-left:auto; margin-right:auto;”}

但此时，我已进入“写不完作业的第二重境界”，称为“确实作业多而写不完作业”，我的成绩也有明显的进步，并且进步之后稳定了下来。由于在第一重境界时培养的心理基础，我并没有因为写不完作业罚站挨批而陷入内耗，而是开始逐渐思考如何对我的时间安排进一步优化。

在第二重境界中，我心安理得完不成作业的心理基础被进一步加强。

高一时，我的物理老师给同学们一句相当有哲理的方法论：“你们只需要完成任务的百分之八十即可。”这句话不仅作为我高中平衡作业和自学的指导方针，而且在我后面竞争更加激烈的大学学习生活中也依然发挥着作用。

{:height=“60%” width=“60%” style=“display:block; margin-left:auto; margin-right:auto;”} 这句话可以理解为另一个版本的“二八定律”（二八定律请自行搜索），你只需要完成任务的百分之八十，就基本掌握了当日任务的全貌，但是却多出了百分之二十的自由安排时间。它打破了完美主义，打破了教条主义，打破了形式主义，让学生有充足的时间来发挥自己的主观能动性。

第三重境界
#

到了高二，我基本能够完成数学、物理的课外题，并且能够完成相当多的英语、化学、生物课外题。这并非因为我牺牲了更多写作业的时间，而是我的成绩提升给我带来了更高的效率。实际上此时的我已经能够轻松完成老师布置的所有作业，但我还是选择贯彻我的二八定律。

我已逐渐进入“不写作业的第三重境界”，即“能写完作业但是选择不写完”。衡量是否进入第三重境界的一个标准就是，个人是否已经掌握了选择权。

{:height=“60%” width=“60%” style=“display:block; margin-left:auto; margin-right:auto;”} 在第三重境界下，学生仍可以进一步提升。

我在高二下学期获得了备考中科大少年班的机会，可以提前参加一次高考。我当时对此非常重视，停了大概三个月的课来备考，开始刷高三的数学题和理综题，并且跟随高三学生参加模拟考试。

第一次模拟考我只有480分，但二模我就复习到了620分，最终高考以604分收尾。以中科大少年班的难度和我当时的水平，考不上是理所应当。

但是，那段备考经历不仅再度提升了我的学习计划制定能力和自学能力，燃烧起了充足的激情，而且切实从分数上实现了提升。长期刷高三题目的我再次看到高二的期末考试题感觉易如反掌，尽管带病参加考试，也还是轻松拿下了那次考试的全市第一名，高三全省模拟考也是全市第一，全省前二百名。

走火入魔
#

以上这些正向进步也不是一直发生，境界深入之后也会走火入魔。

高三下学期由于YQ影响，所有学生居家三个月。这三个月期间，我基本没有跟网课，一直在自己刷题。然而，人是一种社会动物，长期离群索居势必会出现问题。孤独地自学和越来越少的正反馈让我的激情逐渐减退，成绩也开始退步。直到开学之后，我的状态才开始慢慢好转，最后进入复旦，也算是幸运。

所以，任何事物都具有两面性，真理就蕴含在矛盾的对立统一中。

{:height=“60%” width=“60%” style=“display:block; margin-left:auto; margin-right:auto;”}

二、详解“不写作业”
#

如何理解本文“不写作业”的含义
#

本文的“不写作业”，实际上指的是“不在规定时间内写完所有作业”，而不是因为贪玩、懒惰、抗拒或抑郁心理完全不写。在常规情况下，作者的“不写作业”，也会完成至少百分之八十的作业。

作者提到的“不写作业”，不是对老师教学管理的对抗，而是根据自己的特点，在老师的教学管理计划的基础上进行微调，使之适应自己的学习节奏，是一种带有个人特色的补充，其根本目的是提高自己的学习能力，而学习能力的提升由分数的提升来体现。本人在高中三年虽然因为未完成作业没少罚站，但与各位任课老师的关系都很好，不仅是因为本人成绩尚可，更是因为老师看到了本人并未闲着。

比如，尽管本人经常牺牲语文作业的时间，但会坚持花时间阅读名著或杂志，所以本人的语文成绩并不算差。而且，到第三重境界时，本人高三一年刷了将近一百套语文题目，实际上早就弥补了过去牺牲的语文作业。我的语文老师深谙这些事情，所以除了必要的罚站以外，老师并未对我的学习态度做太多批判。

{:height=“60%” width=“60%” style=“display:block; margin-left:auto; margin-right:auto;”}

另外，在第二重境界中，我提到了有时候因为课业压力大，确实完不成全部作业，这也是相当正常的。这些同学可以完成作业的百分之八十以上，已经可以掌握课内的绝大部分知识。但是，有的同学并没有意识到这一点，在巨大压力下也没有采取合适的做法来缓解这一情况，错误的表达形式各式各样：有人过度压缩休息时间身体抱恙，有人满怀怨气和老师进行对抗，有人压抑情绪不敢言语最终破防。我觉得都没有必要。

该艺术适合什么同学
#

很多同学难以实践我说的方法论，被老师批评或者罚站带来的心理压力要大于成绩没有进步带来的焦虑，那就不适合采取这种做法。而且通常情况下，这类同学都比较利索，能够按部就班地跟上老师的教学步伐，成绩也很稳定或者稳步提升。放眼未来就业和生活，服从管理的人有适合他们的岗位，特立独行的人也有各自的活法，没有什么好与坏。作者强调的是辩证地看待问题，并且充分发挥主观能动性，而非必须采取我的做法。

还有一类同学，只有在他人管制下才能稳下心来做事情，否则就会心猿意马。对于这类同学，我建议忘掉这篇文章介绍的所有内容，不要尝试突破管制，好好忍两年就过去了。

从本人不写作业的历史中可以发现，本人是一个心理素质较好（脸皮较厚），不囿于规则（不太喜欢完全服从管理）的人，并且具有一定的自律性（拧死理）。所以如果他人对你的评价中包含这些词汇，那我建议你可以尝试一下本文介绍的方法论。

但是，适合本文方法论的同学也不一定有本人的成长路线。“不写作业”只是一个高度概括的行为，具体采取哪些措施还需要同学自己去深入实践与反思。

{:height=“60%” width=“60%” style=“display:block; margin-left:auto; margin-right:auto;”}

三、如何实践不写作业的艺术？
#

如何不惹老师生气？
#

作者认为，不写作业又不惹老师生气，是艺术性的直接体现。

如果你已经决定好要实践这门艺术，首先你要让老师知道，你写不完作业，但也没闲着。雷军在上大学时，为了提前完成专业课学业，需要翘很多课来自学。为了克服点名问题，他除了让同学帮忙点名，还主动去和老师沟通，并且在老师面前一直展现出求知好学的状态。

{:height=“60%” width=“60%” style=“display:block; margin-left:auto; margin-right:auto;”} 破除壁垒的最好方法就是有效沟通，而沟通本身就是一项艺术。如果你和老师关系很好，那可以直接去跟老师商量。不过，大部分人可能并不会提前和老师商量这事，即便是作者的脸皮也没有厚到这种程度。而且，老师也大概率不会认同你————这样就会出现语文老师布置的作业根本没人写的情况。所以，这种事情你最好还是先斩后奏，但是要以正确的方式和时机“斩”和“奏”。

不按时交上作业会不可避免地引发老师的介入，其形式可能是问原因、挨批评、罚站，严重时可能会挨揍。不写作业的严重程度和自己学习状态、其他同学的学习状态都有关。可能你自己一直以来的表现难以得到老师的信任，你和老师说“我这是艺术”估计挺搞笑的；也可能你已经想好了和老师解释的言辞，但当日太多学生没写完作业了，老师情绪很差，也听不进去你的解释。

知道了原因，你就要规避这些原因。首先，你需要先好好表现几天。你不想写哪科的作业，你就先在哪一科的老师面前好好表现几天，比如上课积极回答问题，课下主动找老师问问题，或者帮忙发作业拿东西，让老师知道你的学习状态很好。如果老师某段时间对全班同学都不满意，你就更不要选择这段时间当出头鸟，否则会很惨。

最好的情况是，老师会主动问你“为什么写不完作业”，那正是你陈述个人想法的时候。你要向老师详细介绍你制定的计划。注意，你应该只介绍这个老师的学科的计划部分，你跟英语老师说“我没写完英语作业是在刷物理题”，大概率得到的回答是“英语高考150分，物理100分，你高考不考英语了吗”。你应该先跟老师道歉：“不好意思老师，我这次确实没来得及，课后我一定补完。”实际上你课后补不补，老师过后大概率也忘了。然后跟老师说：“我买了一本英语综合题，打算这周把其他科成绩稳一稳，之后重点提升英语。”最后虚心向老师请教：“老师您看我最近几次成绩，我需要先练习哪个题型比较好？您有什么建议吗？”如此这般，不仅你能达到自己的目的，而且老师也不会太介意。

{:height=“60%” width=“60%” style=“display:block; margin-left:auto; margin-right:auto;”}

如何利用好不写作业的时间？
#

当你决定拿出20%的写作业的时间来自由规划，而且这些自由规划的时间是稳定存在的（即你已经打通了老师这一关），就需要切实利用好这些时间。

首先，你需要明确自己哪一科目比较薄弱，哪一科目成绩较为稳定。如果你觉得所有科都很薄弱，那选一个你觉得相对感兴趣或者相对容易提升的科目。定下来这门科目后，就不要再改变了，防止东抓一把西抓一把，最后自己都混乱了。

然后，你需要及时购买该科目需要的资料，比如一本课外题或者一本知识点资料，节省下的时间就死磕这本题目或资料。此时，你不能再患得患失，不要去想你暂时亏待的这门科目会不会退步，你的目标只有一个，就是完成这本资料的相当的一部分。你不一定要做完，但至少做到别人随意翻看你这本资料，都不会以“没怎么动”来评价。

{:height=“60%” width=“60%” style=“display:block; margin-left:auto; margin-right:auto;”} 20%的时间不算少，但也真不算多，你只能聚焦一门科目，不要想着自己能利用这点时间同时提升多个科目。说到本质上，就是不要有浮躁心理。你要做好认认真真聚焦一门科目，一个月之后成绩不进反退的准备。就玄学来说，只要你做好这样的准备，最后大概率是成功的。

你在安抚好你亏待科目的老师时，也要注意和你优待的科目老师多交流。你的课外题上可能满是你不会的题，看解析也不一定能看懂，此时抓紧厚着脸皮去找老师，一方面及时搞懂不会的题，另一方面也能让这位老师对你刮目相看，未来倾斜给你更多学习的机会。

{:height=“60%” width=“60%” style=“display:block; margin-left:auto; margin-right:auto;”}

四、结语
#

本文以较为学术性的口吻来写，一方面是作者养成了这样的风格，另一方面也是刻意降低本文的可读性。理想情况下，我希望这篇文章只有在读的学生可以读完全文。学习是自己的事情，主观能动性也是个人的财富，是他人无法强行赋予的。本文标题只是个噱头，。要在教学管理和个人安排上寻找一个平衡点，是作者希望高中生思考的问题；而在社会常规和个人特色之间寻找一个平衡点，则是我们每个人需要毕生思考的问题。
